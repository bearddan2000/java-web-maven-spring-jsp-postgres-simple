# java-web-maven-spring-jsp-postgres-simple

## Description
A jsp springboot java maven build,
that connects to postgres database.

## Tech stack
- springboot
  - web
- jsp
- maven
- bootstrap
- jquery
- dataTable

## Docker stack
- postgres
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
